# Photo_Opp
Simular uma ativação interativa em um estande de evento. Interação com uma tela, tira uma foto, visualiza o resultado com uma moldura do cliente e, se aprovar, realiza o download via QR Code.


# Contexto do Projeto
O desafio simula uma ativação interativa em um estande de evento. O participante interage com uma tela (touchscreen ou celular), tira uma foto, visualiza o resultado com uma moldura do cliente e, se aprovar, pode baixar a imagem via QR Code. A experiência é rápida e contínua.


# Fluxo da Aplicação
## Tela Inicial (Touch para iniciar)


1. Tela de Pré-Captura (Botão de capturar)


2. Contagem Regressiva (3s)


3. Foto Capturada + Moldura aplicada (formato 9:16)


4. Tela de Revisão (Opção de refazer ou aprovar)


5. Tela Final (QR Code para download da imagem)


6. Ao final, volta automaticamente para a tela inicial.
